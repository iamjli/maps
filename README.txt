
This app allows you to explore Boston and get more information about the colleges and universities in the city.

Basic features:
- Select schools are shown with markers that animate when clicked on
- When clicked, markers also display an information window with a short description taken from the Wikipedia API
- A list of schools is found on the left hand side
- The search box allows users to search for substrings in the school names, and also displays the number of results
- The enter key may be used to complete a search
- Markers are updated along with the search results
- The college names are clickable and trigger an animation and information window on the map
- App features a full screen display of the map, while also sharing the screen with the list of colleges
- Responsive design for mobile: search bar is drawable

How to run:
- Clone this repository or download as zip and unpack
- Open index.html in a web browser
